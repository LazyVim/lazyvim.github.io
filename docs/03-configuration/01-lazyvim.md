# LazyVim

**LazyVim** can be configured in the same way as any other plugin.

For example in `lua/plugins/core.lua`

```lua
return {
  {
    "LazyVim/LazyVim",
    opts = {
      colorscheme = "catppuccin",
    }
  }
}
```
