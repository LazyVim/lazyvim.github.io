## Icons & Colorscheme

Icons & colorscheme can be configured as options for the **LazyVim** plugin.

For example in `lua/plugins/core.lua`

```lua
return {
  {
    "LazyVim/LazyVim",
    opts = {
      colorscheme = "catppuccin",
    }
  }
}
```
